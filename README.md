# Joshuboi77 Tap

## How do I install these formulae?

`brew install joshuboi77/tap/<formula>`

Or `brew tap joshuboi77/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "joshuboi77/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
